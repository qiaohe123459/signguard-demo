Token Core 设想接入

原型中将 Token Core 作为底层钱包能力：负责账户体系、地址生成、交易构建、签名前数据解析和签名流程。SignGuard 位于签名前一层，只负责把交易内容转成用户能理解的风险提示。

提交用简介

SignGuard 是一个面向钱包用户的签名前风险提示原型。它会在用户授权、转账、签名之前，用直白语言解释当前操作可能产生的影响，例如是否涉及无限授权、资产转移、合约调用或可疑地址，帮助普通用户减少误签和钓鱼风险。
